# AeroDraw

**AeroDraw** — инструмент для рисования поверх экрана в стиле **Windows 11 Fluent Design**.  
Позволяет рисовать, выделять и аннотировать прямо поверх любых открытых приложений.

---

## Скачать

Перейдите на страницу [Releases](https://github.com/FlowMasterCode1/AeroDraw/releases) и скачайте последнюю версию:

- **AeroDraw-Setup.exe** — установщик (NSIS)
- **AeroDraw-Portable.exe** — портативная версия (без установки)

**Инструкция:** скачал `.exe` → запустил → приложение появится поверх всех окон.

## Возможности

### Инструменты рисования
| Инструмент    | Описание                                              |
|---------------|-------------------------------------------------------|
| Карандаш      | Свободное рисование тонкой линией                     |
| Маркер        | Рисование с прозрачностью 70%                         |
| Выделитель    | Широкое выделение, прозрачность 30%, квадратный конец |
| Ластик        | Стирание нарисованного (destination-out)              |

### Фигуры
- Прямоугольник, Круг/Эллипс, Треугольник, Ромб, Звезда, Шестиугольник
- Прямая линия
- Стрелка с наконечником

### Прочие инструменты
- **Текст** — клик на canvas, вводите текст, Enter — рисует
- **Пипетка** — берёт цвет с canvas
- **Курсор** — режим сквозных кликов (кликаете сквозь окно)
- **Глаз** — скрыть/показать рисунок

### Действия
- **Отмена/Повтор** — до 50 шагов в истории
- **Очистить** — очищает весь canvas
- **Скриншот** — сохраняет рисунок как PNG

### Настройки
- **Палитра** — 24 предустановленных цвета + произвольный цвет
- **Толщина** — от 1 до 40 px со слайдером и превью

## Горячие клавиши

### Инструменты
| Клавиша | Действие         |
|---------|------------------|
| `P`     | Карандаш         |
| `M`     | Маркер           |
| `G`     | Выделитель       |
| `E`     | Ластик           |
| `S`     | Фигуры           |
| `L`     | Линия            |
| `A`     | Стрелка          |
| `T`     | Текст            |
| `I`     | Пипетка          |
| `V`     | Курсор           |
| `H`     | Скрыть/показать  |
| `B`     | Толщина          |
| `C`     | Цвет             |

### Действия
| Клавиша    | Действие             |
|------------|----------------------|
| `Ctrl+Z`   | Отменить             |
| `Ctrl+Y`   | Повторить            |
| `Del`      | Очистить canvas      |
| `Ctrl+S`   | Сохранить скриншот   |
| `[`        | Уменьшить толщину    |
| `]`        | Увеличить толщину    |

### Глобальные хоткеи (работают в фоне)
| Клавиша          | Действие                                          |
|------------------|---------------------------------------------------|
| `Ctrl+Shift+D`   | Переключить режим: рисование ↔ клики сквозь окно |
| `Ctrl+Shift+Q`   | Выйти из AeroDraw                                 |

## Сборка из исходников

### Требования
- [Node.js](https://nodejs.org/) v18+
- [Git](https://git-scm.com/)

### Шаги
```bash
git clone https://github.com/FlowMasterCode1/AeroDraw.git
cd AeroDraw
npm install
npm start        # запуск в режиме разработки
npm run build    # сборка .exe (Windows)
```

Собранный файл появится в папке `dist/`.
